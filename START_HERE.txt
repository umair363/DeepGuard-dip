🎯 WHAT THIS PROJECT DOES:
============================

Detects AI-generated fake faces using:

1. SPATIAL BRANCH
   └─ Analyzes RGB images for visual artifacts
   └─ Detects: blending issues, texture problems, skin tone artifacts

2. FREQUENCY BRANCH
   └─ Analyzes FFT spectrum for AI patterns
   └─ Detects: frequency domain anomalies left by AI tools

3. FUSION CLASSIFIER
   └─ Combines both branches for final prediction
   └─ Accuracy: 88-92% (depends on data size)

🐛 COMMON ISSUES & QUICK FIXES:
================================

Issue: "No module named src_preprocessing"
→ Make sure you're in the project root directory

Issue: "CUDA not found"
→ This is OK! CPU will be used automatically

Issue: Out of Memory
→ Reduce batch_size in config.yaml (try 8 or 4)

Issue: "Model not found" during inference
→ Train the model first: python train_model.py

Issue: Slow preprocessing
→ Normal on first run. Results are cached.

Issue: "No face detected"
→ Ensure faces are clear and frontal
